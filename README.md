### Nork-Town (cidade)
- 1 Pessoa pode ter até 3 veículos
- os veiculos podem ter as cores 'amarelo', 'azul' ou 'cinza'.
- os veiculos podem ser dos modelos: 'escotilha', 'sedan' ou 'conversível'.
